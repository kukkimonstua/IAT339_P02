Colour Scheme

The light cyans will be used for background decoration.

The greens will be used for links and buttons.

The muted oranges will be used to showcase whether an element is interactive or not.

The browns are used to showcase the interactive elements have been selected.

The web-pages will use black, large text to keep the pages visable.
